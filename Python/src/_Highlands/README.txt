To setup users and tables (you must have a root and manager user):
	setup.xlsx

To edit questions:
	questions.xlsx

To initilize the system:
	python initialize_database.py

To run the server:
	python server.py
	
To start the client in a browser:
	http://<server-ip>:<port>/client.html
